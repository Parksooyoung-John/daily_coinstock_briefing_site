텍스트로 먼저 요약을 받고, 이미지 리포트로 주요 지수, 보유자산 영향, 뉴스와 리스크를 다시 확인합니다. 모바일에서 바로 읽을 수 있도록 1080px 폭 기준으로 생성합니다.

Service Options

계좌 연동 방식만 선택하면 됩니다

초기 상품은 발송 채널과 이미지 템플릿을 단순화했습니다. 텔레그램으로 고정해 전달 품질과 운영 안정성을 우선합니다.

증권사 API 연동

해외주식 보유현황, 평가금액, 손익, 수익률을 기반으로 브리핑합니다. 증권사 API 제공 여부를 먼저 확인합니다.

업비트 연동

보유 코인, 원화 예수금, 종목별 수익률과 주요 암호화폐 뉴스를 함께 정리합니다.

계좌 미연동

API 연동 없이 관심 종목, 시장 지표, 뉴스 중심으로 데일리 브리핑을 제공합니다.

Setup Flow

구축은 4단계로 진행합니다

  1. 고객 정보 확인계좌 연동 방식, 텔레그램 Bot Token, Chat ID를 확인합니다.
  2. 서버/API 설정AWS 서버에 고객별 config와 secrets env를 분리해 설정합니다.
  3. 테스트 발송샘플 리포트와 실제 데이터 리포트를 발송 전 검토합니다.
  4. 자동 발송 등록매일 오전 9시 텔레그램 자동 발송 cron을 등록합니다.

보안 원칙

API 키는 조회 권한만 사용합니다. 주문, 매수, 매도, 출금 권한은 사용하지 않습니다. 고객별 토큰은 서버의 secrets 파일에만 저장합니다.

투자 유의사항

리포트는 투자 참고용 정보입니다. 투자 판단과 결과의 책임은 사용자 본인에게 있습니다.
