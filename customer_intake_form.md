# 고객 정보 요청서

MVP 상품은 텔레그램으로 텍스트 리포트와 이미지 리포트를 발송합니다. 카카오톡 발송은 초기 상품 범위에 포함하지 않습니다.

## 1. 기본 정보

```text
고객명:
연락처:
희망 발송 시간: 오전 9시 기본
리포트 목적: 개인 투자 현황 확인 / 시장 뉴스 확인 / 기타
```

## 2. 계좌 연동 방식 선택

아래 중 하나를 선택합니다.

```text
[ ] 증권사 API 연동
[ ] 업비트 연동
[ ] 계좌 미연동
```

### 증권사 API 연동

고객에게 요청할 정보:

```text
증권사명:
API 제공 여부 확인:
API 앱키:
API 시크릿:
계좌번호:
상품코드:
AWS 서버 IP 허용 등록 여부:
조회 권한만 발급했는지:
```

주의:

- 증권사에 따라 API 연동이 불가능할 수 있습니다.
- 주문/출금 권한은 사용하지 않습니다.

### 업비트 연동

고객에게 요청할 정보:

```text
UPBIT_ACCESS_KEY:
UPBIT_SECRET_KEY:
AWS 서버 IP 허용 등록 여부:
권한이 자산조회 전용인지:
```

주의:

- 출금 권한은 절대 허용하지 않습니다.
- 허용 IP에는 운영 서버 공인 IP만 등록합니다.

### 계좌 미연동

고객에게 요청할 정보:

```text
관심 종목 티커:
관심 시장: 미국 / 한국 / 코인 / 환율
관심 뉴스 키워드:
```

제공 범위:

- 보유현황 자동 분석은 제공하지 않습니다.
- 시장/뉴스/관심종목 중심 브리핑을 제공합니다.

## 3. 텔레그램 정보

고객에게 요청할 정보:

```text
TELEGRAM_BOT_TOKEN:
TELEGRAM_CHAT_ID:
고객이 봇에게 /start 전송 여부:
```

안내 문구:

```text
이미지 리포트 품질을 위해 텔레그램 기반으로 제공됩니다.
텔레그램에서는 텍스트 브리핑과 이미지 리포트가 같은 채팅방에 바로 도착합니다.
```

## 4. 동의 확인

고객에게 아래 내용을 확인받습니다.

```text
[ ] 본 서비스는 투자 정보 제공용이며 투자 권유가 아님을 확인했습니다.
[ ] 투자 결정의 책임은 본인에게 있음을 확인했습니다.
[ ] API 키는 조회 전용 권한으로만 발급했습니다.
[ ] 주문, 매도, 매수, 출금 자동화는 요청하지 않습니다.
[ ] 텔레그램 봇으로 리포트를 받는 것에 동의합니다.
```
